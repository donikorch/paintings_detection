# paintings_detection
Применение технологий обработки изображений для цифровой атрибуции

При нахождении автора и стиля картины использовались такие признаковые пространства, как HOG, HoughLines и GaborFilter.

Стили:
  1.	Импрессионизм
  2.	Постимпрессионизм
  3.	Кубизм
  4.	Супрематизм
  5.	Сюрреализм
Авторы:
  1.	Винсент Ван Гог
  2.	Леонардо да Винчи
  3.	Пабло Пикассо
  4.	Сальвадор Дали
  5.	Эдвард Мунк
